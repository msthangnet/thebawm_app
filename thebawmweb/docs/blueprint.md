# **App Name**: The Bawm

## Core Features:

- User Authentication: Allow users to create accounts, log in, and manage their profiles using Firebase Authentication.
- Profile Creation: Enable users to create and customize their profiles with profile picture, cover image, username, and bio, and edit user information like Hometown, Live in, Current study, Relationship status, and Gender.
- Social Feed: Display a social feed of posts from all users, similar to a Facebook feed. Users can like, comment on, and share posts.
- Post Creation: Enable users to create posts with text, attach images and videos, and share them on the social feed.
- User Timeline: Display a user's posts on their profile page, similar to a Facebook user timeline.
- Landing Page Generator: Uses a generative AI tool to analyze popular social media trends, and suggests some re-wording of the title, and other taglines on the landing page.
- Account Settings: Provide users with account settings to update their password and delete their account.

## Style Guidelines:

- Primary color: Vivid blue (#29ABE2) to convey trust and connection.
- Background color: Light gray (#F0F2F5) for a clean and modern look.
- Accent color: Orange (#F26419) to highlight CTAs and important information.
- Body and headline font: 'Inter', a sans-serif font, will be used throughout the application.
- Forms (login and sign-up) will be designed within a visually distinct box to focus user attention.
- Use clean and modern icons for navigation and social interactions (likes, comments, shares).
- Subtle animations for loading states and user interactions to enhance user experience.
